# Week 1: 系统设计基础概念

## 📚 本周学习目标

- 理解系统设计的核心原则
- 掌握系统设计面试的答题框架
- 学习可扩展性、可靠性、可用性的概念
- 理解 CAP 定理及其应用
- 掌握性能指标的计算方法

## 📖 学习内容

### 1. [系统设计面试流程](./01-interview-process.md)
- 如何理解需求
- 如何估算容量
- 答题框架和步骤

### 2. [可扩展性（Scalability）](./02-scalability.md)
- 垂直扩展 vs 水平扩展
- 扩展的瓶颈
- 如何设计可扩展的系统

### 3. [可靠性与可用性](./03-reliability-availability.md)
- 可靠性（Reliability）
- 可用性（Availability）
- 容错设计
- SLA/SLO/SLI

### 4. [性能指标](./04-performance-metrics.md)
- 延迟（Latency）
- 吞吐量（Throughput）
- QPS/TPS
- 响应时间

### 5. [CAP 定理](./05-cap-theorem.md)
- 一致性（Consistency）
- 可用性（Availability）
- 分区容错性（Partition Tolerance）
- 权衡取舍

### 6. [一致性模型](./06-consistency-models.md)
- 强一致性
- 最终一致性
- 因果一致性
- 会话一致性
